[nvm-windows](https://github.com/coreybutler/nvm-windows) nodejs版本管理工具

> 查看已安装的Node.js版本：nvm ls
> 安装特定版本的Node.js：nvm install 20.12.2
> 切换到特定版本的Node.js：nvm use 20.12.2
> 卸载不需要的Node.js版本：nvm uninstall 20.12.2
